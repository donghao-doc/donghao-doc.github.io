# PixiJS

PixiJS 是一个超快的 2D **渲染引擎**，它拥有强大的图片渲染能力和场景图技术，可用于制作动画、炫酷的交互、简单的小游戏等。

:::warning

PixiJS 不是游戏引擎。

:::

## 安装和使用

```bash
npm install pixi.js
```

```ts
import * as PIXI from 'pixi.js';
```
